Welcome to Capsule Story Wiki

About Capsule Story

Capsule Story is a unique hybrid game that combines the best elements of Minecraft and Pokemon in a top-down 2D format. This wiki serves as the ultimate guide for players to learn about game mechanics, strategies, and community contributions.

Getting Started

Welcome to the world of Capsule Story! This guide will help you begin your journey as a monster tamer and master builder.

Installation

  • Download the appropriate version for your platform (Android or Desktop)
  • For Android: Install the APK file
  • For Desktop: Extract the ZIP and run the JAR file

First Steps

  1. Create your character
  2. Learn basic controls
  3. Gather resources
  4. Build your first shelter
  5. Encounter your first monster

Core Game Mechanics

Crafting

Combine resources to create tools, items, and building materials.

Building

Create structures using various blocks and materials.

Monster Taming

Find, battle, and tame unique monsters throughout the world.

World Exploration

Explore procedurally generated worlds with unique biomes.
